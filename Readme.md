## How to execute python script

You have two ways to execute python code on your computer. First case:
```
python3 <filename>.py <args>
```
Second case:
```
chmod +x <filename>.py
./<filename>.py <args>
```
ATTENTION. In the secon case the first line in your python script must be
```
#!/usr/bin/env python3
```
